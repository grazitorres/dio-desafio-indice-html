Introdução ao HTML na Prática

Voltar

HTML é uma linguagem de marcação usada para estruturar o conteúdo de páginas web.
A estrutura do HTML é organizada por tags e atributos (Immagem 1):

  • As tags definem os elementos da página (como <p> para parágrafo ou <h1> para título), enquanto os atributos modificam ou complementam esses elementos (por exemplo, href em um link ou src em uma imagem).
  • O elemento <head> é responsável por armazenar informações que não são exibidas diretamente na página, mas que são importantes para o funcionamento e interpretação do site pelos navegadores. Nele, colocamos os metadados da página, como a codificação de caracteres, o título da aba do navegador, o link para o arquivo CSS externo e também o link para fontes externas, como as do Google Fonts.
  • O elemento <body> é onde fica todo o conteúdo visível da página, como textos, imagens, listas, links, botões e outros elementos que o usuário irá visualizar e interagir.
