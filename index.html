Introdução ao HTML na Prática

Voltar

HTML é uma linguagem de marcação usada para estruturar o conteúdo de páginas web.
A estrutura do HTML é organizada por tags e atributos:

  • As tags definem os elementos da página (como <p> para parágrafo ou <h1> para título), enquanto os atributos modificam ou complementam esses elementos (por exemplo, href em um link ou src em uma imagem).
  • O elemento <head> é responsável por armazenar informações que não são exibidas diretamente na página, mas que são importantes para o funcionamento e interpretação do site pelos navegadores. Nele, colocamos os metadados da página, como a codificação de caracteres, o título da aba do navegador, o link para o arquivo CSS externo e também o link para fontes externas, como as do Google Fonts.
  • O elemento <body> é onde fica todo o conteúdo visível da página, como textos, imagens, listas, links, botões e outros elementos que o usuário irá visualizar e interagir.

Nos próximos tópicos, vamos abordar as tags e os principais atributos relacionados ao HTML. Além disso, entenderemos um pouco sobre a estrutura e a semântica no HTML.

Trabalhando com Formulários em HTML

Voltar
  1. <form>
    • Define o formulário
    • Atributos mais comuns:
      • action :URL para onde os dados do formulário serão enviados.
      • method : método de envio (GET ou POST).

  2. <input>
    • Campo de entrada multifuncional (texto, senha, e-mail, número, checkbox, radio etc.).
    • Atributos importantes:
      • type : tipo do input (text, password, email, number, checkbox, radio, etc.).
      • name : nome do campo (usado no backend).
      • value : valor enviado.
      • placeholder : texto de dica dentro do campo.
      • requered : torna o campo obrigatório.

  3. checkbox (<input type="checkbox">)
    • Um campo de seleção com formato quadrado, que o usuário pode marcar ou desmarcar.
    • Serve para permitir múltiplas escolhas. Ou seja, o usuário pode marcar quantas opções quiser.

  4. radio (<input type="radio">)
    • Um campo de seleção em formato circular, que permite escolher apenas uma opção dentro de um grupo.
    • Usado quando o usuário deve escolher só uma alternativa entre várias (exclusão mútua).
    • Para que funcione corretamente, todos os radio buttons de um mesmo grupo devem ter o mesmo name.

  5. <textarea>
    • Área para texto longo (como comentários).
    • Atributos comuns:
      • name :
      • rows e cols: tamanho da caixa
      • placeholder
      • requered

  6. <select> e <option>
    • Juntas, essas tags criam um menu suspenso (também chamado de lista drop-down), onde o usuário pode escolher uma opção entre várias.
    • Usado quando você quer apresentar várias opções, mas sem ocupar muito espaço na tela.
    • Como funciona:
      • a tag <select> cria o campo de seleção.
      • a tag <option> define cada uma das opções disponíveis dentro do menu.
  7. <fieldset> e <legend>
    • <fieldset>: é usada para agrupar visualmente elementos relacionados em um formulário. Ela cria uma caixa com borda ao redor dos campos, ajudando a organizar as informações e deixar o formulário mais claro para o usuário.
    • <legend>: Usado dentro de <fieldset> para dar um título àquele grupo.

Estruturando seu HTML + Formatações

Voltar

  1. Títulos <h1> até <h6>

    • Representam títulos e subtítulos.
    • <h1> é o mais importante (maior), <h6> o menos importante (menor).
    • Usados para organizar o conteúdo por hierarquia.
    • Exemplos:

      Título em h1

      Título em h2

      Título em h3

      Título em h4

      Título em h5
      Título em h6

  2. Formatação de texto

    • <strong>: Deixa o texto em negrito com ênfase semântica (importante para leitores de tela).
    • <b>: Deixa em negrito, mas sem significado semântico.
    • <i>: Texto em itálico com ênfase semântica.
    • <u>: Texto sublinhado.
    • <sub>: Texto em subscrito (abaixo da linha). Ex: H2O
    • <sup>: Texto em sobrescrito (acima da linha). Ex: x2
    • <del>: Mostra o texto riscado riscado, indicando algo removido ou corrigido.
    • <blockquote>: Citação de bloco, usada para trechos de textos ou falas mais longas, geralmente com recuo.
      • Este é um exemplo de blockquote
    • <mark>: usada para destacar uma parte do texto, como se você estivesse usando um marcador(tipo marca-texto).

  3. Organização de layout

    • <div>: Contêiner em bloco, usado para agrupar seções maiores ou estruturar a página.
    • <span>: Contêiner em linha, usado para aplicar estilo ou lógica a um trecho específico de texto.

  4. Tags desatualizadas

    • <font>: Usada para alterar tipo, tamanho ou cor da fonte (hoje se usa CSS).
    • Evite usar, mas ainda funcionam.

  5. Cores no HTML

    No HTML, cores podem ser definidas de três formas principais:

    • Nome da cor:
      • Usa nomes reconhecidos pelo navegador.
      • Exemplo: red, blue, green, black, white.
    • Hexadecimal:
      • Começa com # e é seguido por 6 dígitos/letras.
      • Representa a combinação de vermelho, verde e azul (RGB).
      • Exemplo: #FF0000 é vermelho.
    • RGBA (Red, Green, Blue, Alpha):
      • Usa valores entre 0 e 255 para R, G, B.
      • O "A" (alpha) representa a opacidade (de 0 = transparente até 1 = opaco).

Trabalhando com Mídias utilizando HTML

Voltar
  1. <img>
    • Usada para exibir imagens.
    • Tag autossuficiente (não tem fechamento).
    • Atributos mais comuns:
      • src: caminho da imagem (obrigatório).
      • alt : texto alternativo caso a imagem não carregue (obrigatório para acessibilidade).
      • width/height : define o tamanho (opcional).
      • title : serve para mostrar um texto quando o usuário passa o mouse sobre a imagem. Esse texto aparece como uma caixinha flutuante, tipo uma "dica" ou "informação extra".

  2. <audio>
    • Usada para incorporar áudio.
    • Pode conter a tag <source> para oferecer diferentes formatos de arquivo.
    • Dentro da tag, escreva um aviso como "Seu navegador não tem suporte para este áudio " caso a pessoa esteja usando um navegador antigo e o áudio não funcione.
    • Atributos/tags mais comuns:
      • src: caminho da áudio (pode ser usado direto na tag ou via <source>).
      • controls : exibe os controles de reprodução (play, pause, etc.).
      • autoplay : começa a tocar automaticamente (não recomendado sem interação).
      • loop : repete o áudio.
      • muted : começa mudo.
    • Observação: Usar o src direto é simples, mas limitado a um único formato. Já o uso de <source> é mais recomendado, pois permite compatibilidade com diferentes navegadores e ainda dá a opção de mostrar uma mensagem se o áudio não for suportado.

  3. <video>
    • Usada para incorporar vídeos.
    • Também pode usar a tag <source> e colocar a mensagem de aviso caso o navegador não tenha suporte para o vídeo.
    • Atributos mais comuns:
      • src: caminho do vídeo.
      • controls : mostra os botões de controle.
      • width/height : define o tamanho do vídeo.
      • autoplay, loop, muted, poster (imagem de capa antes de iniciar).

  4. <track>
    • Usada dentro da tag <video> ou <audio> para adicionar legendas, transcrições ou descrições.
    • Atributos importantes:
      • src: caminho do arquivo (.vtt ou .rst).
      • kind : tipo de legenda (subtitles, captions, descriptions, etc).
      • srlang : idioma.
      • label : nome exibido para o usuário.
      • default : define como padrão.

  5. <iframe>
    • Usada para incorporar conteúdo externo, como vídeos do YouTube, mapas, sites, etc.
    • É uma janela dentro da página.
    • Atributos importantes:
      • src: URL do conteúdo incorporado.
      • width/height : dimensões.
      • title : acessibilidade
      • allowfullscreen : permite o conteúdo ser exibido em tela cheia.

  6. <figure> e <figcaption>

    • <figure> é usada para agrupar conteúdo de mídia com sua legenda.
    • <figcaption> é usada dentro de <figure> para adicionar uma legenda descritiva.
    • Ajuda a organizar melhor imagens, vídeos, gráficos ou trechos de código com descrição.

Criando Tabelas com HTML

Voltar

  1. Tags principais de uma Tabela

    • <table>Cria a tabela.
    • <tr> (table row): Representa uma linha da tabela.
    • <td> (table data): Representa uma célula de dados comum dentro de uma linha.
    • <th>(table header): Representa uma célula de cabeçalho, geralmente vem em negrito e centralizada por padrão.

  2. Estrutura semântica com agrupamentos

    • <thead>: Agrupa o cabeçalho da tabela.
    • <tbody>: Agrupa o corpo (conteúdo principal) da tabela.
    • <tfoot>: Agrupa o rodapé, útil para somas, totais, observações.

    Esses elementos ajudam na organização semântica e melhoram o suporte a acessibilidade.

  3. Legenda

    • <caption>: Adiciona um título descritivo para a tabela. Fica geralmente acima da tabela.

  4. Mesclar células

    • colspan="n": Mescla células horizontalmente (colunas).
    • rowspan="n" Mescla células verticalmente (linhas).

Entendendo HTML Semântico

Voltar

O HTML5 trouxe avanços importantes em relação ao HTML4, especialmente no que diz respeito à organização, semântica e acessibilidade do conteúdo nas páginas web. Antes, era comum o uso excessivo de <div> com classes diferentes para representar seções da página, o que prejudicava a acessibilidade e dificultava a leitura do código por leitores de tela. Com a nova versão, surgiram elementos semânticos que deixam o código mais claro e acessível. Além disso, foram definidos padrões como a WCAG (Web Content Accessibility Guidelines) e o uso de atributos ARIA para melhorar a acessibilidade.

  • Principais tags semânticas:

    • <main>: Define o conteúdo principal da página, que é exclusivo e diretamente relacionado ao propósito central daquele documento.
    • <header>: Representa o cabeçalho de uma página ou de uma seção. Pode conter títulos, logotipo, menu de navegação, etc.
    • <footer>: Indica o rodapé da página ou de uma seção. Costuma incluir informações como direitos autorais, links úteis ou dados de contato.
    • <nav>: Utilizado para agrupar links de navegação que apontam para outras partes do site ou para outras páginas.
    • <section>: Agrupa conteúdo tematicamente relacionado. É ideal para dividir uma página em partes significativas, como tópicos ou capítulos.
    • <article>: Representa um conteúdo independente e autocontido, como uma notícia, post de blog ou comentário.
    • <aside>: Indica conteúdo complementar, como uma barra lateral ou uma dica relacionada ao conteúdo principal.
    • <blockquote>: Usado para citações longas, exibidas em bloco. Indica que o conteúdo foi retirado de outra fonte.
    • <q>: Representa citações curtas dentro de uma linha. O navegador normalmente coloca aspas automaticamente.
    • <figure>: Agrupa elementos de mídia (como imagens, gráficos ou vídeos) com legenda.
    • <figcaption>:Tag usada dentro do <figure> para fornecer a legenda ou descrição da mídia.
    • <picture>: Permite oferecer diferentes versões de uma imagem para diferentes tamanhos de tela ou dispositivos, sendo essencial em layouts responsivos.
