Introdução ao HTML na Prática

Voltar

HTML é uma linguagem de marcação usada para estruturar o conteúdo de páginas web.
A estrutura do HTML é organizada por tags e atributos (Imagem 1):

  • As tags definem os elementos da página (como <p> para parágrafo ou <h1> para título), enquanto os atributos modificam ou complementam esses elementos (por exemplo, href em um link ou src em uma imagem).
  • O elemento <head> é responsável por armazenar informações que não são exibidas diretamente na página, mas que são importantes para o funcionamento e interpretação do site pelos navegadores. Nele, colocamos os metadados da página, como a codificação de caracteres, o título da aba do navegador, o link para o arquivo CSS externo e também o link para fontes externas, como as do Google Fonts.
  • O elemento <body> é onde fica todo o conteúdo visível da página, como textos, imagens, listas, links, botões e outros elementos que o usuário irá visualizar e interagir.

Nos próximos tópicos, vamos abordar as tags e os principais atributos relacionados ao HTML. Além disso, entenderemos um pouco sobre a estrutura e a semântica no HTML.

Trabalhando com Formulários em HTML

Voltar
  1. <form>
    • Define o formulário
    • Atributos mais comuns:
      • action :URL para onde os dados do formulário serão enviados
      • method : método de envio (GET ou POST).

  2. <input>
    • Campo de entrada multifuncional (texto, senha, e-mail, número, checkbox, rádio etc.).
    • Atributos importantes:
      • type : tipo do input (text, password, email, number, checkbox, radio, etc.).
      • name : nome do campo (usado no backend).
      • value : valor enviado
      • placeholder : texto de dica dentro do campo.
      • requered : torna o campo obrigatório.

  3. checkbox (<input type="checkbox">)
    • Um campo de seleção com formato quadrado, que o usuário pode marcar ou desmarcar.
    • Serve para permitir múltiplas escolhas. Ou seja, o usuário pode marcar quantas opções quiser.

  4. radio (<input type="radio">)
    • Um campo de seleção em formato circular, que permite escolher apenas uma opção dentro de um grupo.
    • Usado quando o usuário deve escolher só uma alternativa entre várias (exclusão mútua).
    • Para que funcione corretamente, todos os radio buttons de um mesmo grupo devem ter o mesmo name.

  5. <textarea>
    • Área para texto longo (como comentários).
    • Atributos comuns:
      • name :
      • rows e cols: tamanho da caixa
      • placeholder
      • requered

  6. <select> e <option>
    • Juntas, essas tags criam um menu suspenso (também chamado de lista drop-down), onde o usuário pode escolher uma opção entre várias.
    • Usado quando você quer apresentar várias opções, mas sem ocupar muito espaço na tela.
    • Como funciona:
      • a tag <select> cria o campo de seleção.
      • a tag <option> define cada uma das opções disponíveis dentro do menu.
